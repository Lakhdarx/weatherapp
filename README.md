# Weather App

A minimal React + Vite weather dashboard.

It allows users to search any city and instantly view real-time weather data.

## Features

- 🔎 Search weather by city name
- 🌡 Displays temperature, weather condition, humidity, and wind speed
- 🌍 Real-time data from the Visual Crossing Weather API
- 🎨 Dynamic weather icons based on conditions
- ⚡ Fast, lightweight UI with instant updates

## Tech stack

- React 19
- Vite
- modern-normalize
- ESLint
- Visual Crossing Weather API

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Project structure

- `src/App.jsx` — app entry component
- `src/components/Weather.jsx` — weather dashboard
- `src/components/SearchBar.jsx` — city search form
- `src/styles/` — CSS styling
